MochiMart is a full-stack shopping application built using Flask, HTML, CSS, and Bootstrap.
It allows users to browse products, add items to their cart, and complete a simulated purchase.
The backend exposes a lightweight REST API for product management, while the frontend provides a clean and responsive shopping interface.

---

# Tech Stack
*Frontend* : HTML5,CSS3,Bootstrap

*Backend* : Flask (Python),Jinja2,Flask-CORS

*API Format* : REST (JSON)

---

# Features

*User Features*

- Browse and view available products

- Add products to the cart

- Remove items from the cart

- View total cost before checkout

- Proceed to a simulated purchase

- Fully responsive interface for mobile and desktop users
